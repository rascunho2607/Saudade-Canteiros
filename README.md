# Saudade & Canteiros

Uma experiência visual em 3D inspirada em uma árvore como caminho de lembranças, saudade e reencontro.

## O que é

Este projeto é uma página web interativa que combina:
- uma cena 3D em low poly;
- uma narrativa em forma de galhos e memórias;
- fotos e textos que aparecem conforme a experiência avança;
- elementos visuais com uma estética poética e delicada.

## Estrutura do projeto

- `index.html` — página principal com a cena, a narrativa e os controles;
- `src/audio/` — arquivos de áudio usados na experiência;
- `src/fotos/` — imagens utilizadas nas memórias.

## Como abrir

1. Abra o arquivo `index.html` em um navegador moderno.
2. Se a página não carregar corretamente, verifique se há conexão com a internet na primeira abertura, pois a cena 3D usa a biblioteca Three.js via CDN.

## Funcionalidades

- rolagem para explorar a árvore em perspectiva;
- abertura de cartões de memória ao clicar nos galhos;
- modo de demonstração para visualizar todos os galhos;
- som leve opcional;
- interface com ajustes visuais e de movimento.

## Observação

Este projeto é uma apresentação artística e pode ser ajustado livremente para novas histórias, fotos ou sons.
